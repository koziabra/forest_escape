# Forest Escape — контекст проекта для Claude Code

## Кратко
3D-игра от первого лица в жанре ночного хоррор-выживания на **Godot 4.7 (GDScript)**.
Игрок бродит по тёмному лесу, собирает светящиеся осколки звезды, избегает оборотней,
светит фонарём (но свет выдаёт его хищникам). Собрав все осколки — доходит до
финальных ворот, чтобы победить.

Автор: **koziabra**. Пользователь — начинающий разработчик, объясняй шаги просто,
общение и текст в игре — **на русском языке**.

## Целевые платформы
Итог должен запускаться на: **Windows, Android, Web (HTML5)**. Рендерер — **Mobile**
(совместим со всеми тремя и лёгок для слабых устройств).

## Как устроен проект (важно)
- Всё строится **процедурно в коде** (GDScript), без хрупких .tscn.
- Главная сцена `Main.tscn` — это один корневой `Node3D` со скриптом `game.gd`.
- `game.gd` в `_ready()` создаёт окружение, свет, землю, деревья, осколки,
  оборотней, ворота, игрока и интерфейс.
- Управление — тач (левая половина экрана: идти, правая: смотреть) + WASD для ПК.

## Текущая реализация (файл game.gd уже в репозитории)
Уже сделано:
- Ночь: WorldEnvironment (тёмный фон, туман), тусклый лунный DirectionalLight3D.
- Земля (PlaneMesh) + процедурные текстуры (NoiseTexture2D).
- ~900 деревьев через MultiMesh (быстро): 3 вида — сосны, дубы, берёзы.
- 5 осколков-кристаллов (светятся, вращаются, парят).
- 6 оборотней (модель: туловище, голова, морда, уши, 4 лапы, хвост, красные глаза),
  бродят и преследуют; видят игрока дальше при включённом фонаре.
- Фонарь (SpotLight3D) с кнопкой в интерфейсе.
- Финальные ворота: собрал все осколки → ворота загораются → дойди до них → победа.
- Меню старта и экран окончания (победа/проигрыш) с кнопками.
